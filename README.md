## 👨‍💻 О себе

**Власов Дмитрий** | Группа **ПИНб-31**

Этот репозиторий создан в рамках курса по программной инженерии. Здесь собраны лабораторные работы и проекты, демонстрирующие углубленное изучение JavaScript и смежных технологий, а также применение принципов программной инженерии на практике.

## 🗂️ Структура проектов

Название | Технологии | Краткое описание
---|---|---
**[📁 part-1](./part-1)** | JavaScript, Node.js, Express | **Основы JavaScript и серверная разработка**: от введения в JS и задач на Stepik до работы с Node.js, event loop, созданием HTTP/Express серверов, обработкой массивов, JSON, синхронными запросами, статическими и динамическими парсерами.
**[📁 part-2](./part-2)** | JavaScript, SQLite, EJS | **Продвинутый JavaScript и бэкенд-разработка**: регулярные выражения, структуры данных, ООП/классы, работа с SQLite, шаблонизатор EJS, паттерн MVC, хеширование и аутентификация, промисы, многопоточность.

## 🛠️ Технологический стек

*   **Язык:** JavaScript (ES6+)
*   **Среда выполнения:** Node.js
*   **Инструменты:** Git

## 📊 Статус репозитория

## 🚀 Быстрый старт

Чтобы клонировать и начать работу с любым из проектов локально, выполните следующие шаги:

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/Dix1k/SE-25-JavaScript.git
    cd SE-25-JavaScript
    ```

2.  **Перейдите в интересующую вас директорию проекта** (например, `part-1` или `part-2`).

3.  **Следуйте инструкциям внутри конкретного проекта** (если они есть в его локальном `README.md` или комментариях в коде) для установки зависимостей и запуска.
